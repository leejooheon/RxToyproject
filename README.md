## 스터디 목표
- Kotlin 언어를 학습한다.
- github API를 이용한 간단한 프로젝트를 통해 주요 기술들을 순차적으로 적용해보며 이해를 돕는다.
- Rx → MVVM → Clean Architecture를 스터디하며 순차적으로 적용해본다.
- 회사 프로젝트에 적용할 수 있는 수준의 공통 Framework를 구축한다.
